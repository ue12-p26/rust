# Comments

Rust uses the same comments style as C/C++: `//` and `/* ... */`.

Example with `//`:

```{code-cell} rust
:tags: [skip-execution]
:class: disabled
// A C++ style comment.
```

Example with `/* ... */`:

```{code-cell} rust
:tags: [skip-execution]
:class: disabled
/* A C style comment. */

/* A C style comment
   on multiple
   lines. */
```
