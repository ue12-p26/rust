# About Rust

Rust ([official web site](https://rust-lang.org/)), now managed by the
[Rust Foundation](https://rustfoundation.org/), was created by a Mozilla's
employee (Graydon Hoare) in 2006.

Like the C language, Rust initially targeted system programming. It has been
adopted by major tech companies like Discord, Microsoft, Facebook, Dropbox,
Firefox, Cloudflare
([Rust use cases](https://kruschecompany.com/rust-programming-language-use-cases/)),
and is used in browsers, kernels, block chain and infrastructure components.
This provides Rust with a strong basis of supporters and contributors.

Now it is more seen, like C, as a general-purpose programming language.
Its strong points are:

- Performance.
- Type safety.
- Concurrency.
- Memory safety.
