# `array` methods

:::{warning} In progress
This section is still being written.
:::

We list here some of the methods of the `array` type. For a full list,
see [Primitive Type `array`](https://doc.rust-lang.org/std/primitive.array.html)
inside the official Rust reference documentation.

:::{list-table} Some methods of `array`
:name: tab-array-methods
:header-rows: 1
:align: center

* - Method
  - Description
* - `each_mut()`
  - Returns an array of mutable references on elements.
* - `each_ref()`
  - Returns an array of references on elements.
* - `map(f)`
  - Applies function `f()` on each element of the array and returns an
    array of the same size.
:::
