# `Vec` methods

:::{list-table} Some methods of `Vec`
:name: tab-vec-methods
:header-rows: 1
:align: center

* - Method
  - Description
* - `clear()`
  - Removes all elements.
* - `insert()`
  - Inserts an element at a position.
* - `is_empty()`
  - Returns `true` if the vector contains no elements.
* - `len()`
  - Returns the number of elements.
* - `new()`
  - Creates a new vector.
* - `pop()`
  - Removes the last element and returns it.
* - `push()`
  - Appends an element.
* - `remove()`
  - Removes an element at a position.
* - `resize()`
  - Resizes vector, appending copies of a specified element if required.
:::
