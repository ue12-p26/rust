# Organisation

This book has been primarily written for an oral presentation, though
we have tried to make it as usable as possible for a standalone
reading.

The course starts with a presentation of the Rust environment, which
explains how to install Rust, how to use the compiler `rustc`, and how
to use `cargo`, the Rust project manager application.

Then it is divided into 5 parts, each representing a level of
progression:

- Basics
- Fundamentals
- Proficiency
- Expertise
- Mastery

The various subjects of the course are thus sorted into these 5 parts
according to their assumed level of difficulty and their
inter-dependency.
Some subjects being too complex to treat in a single chapter (e.g.:
memory, functions, types) are split in multiple chapters spread across
the 5 parts/levels.

At the end of the book are Appendices, in which you will find:

- Tables of methods for various objects, macros, etc.
- Solutions to the exercises.
- A Bibliography.
