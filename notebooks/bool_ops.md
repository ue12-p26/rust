# Boolean operators

Boolean values can be combined using the logical operators presented in
{numref}`tab-logical-op`.
The operators in {numref}`tab-comp-op` always return a boolean value.

:::{list-table} Logical operators
:name: tab-logical-op
:header-rows: 1
:align: center

* - Operator
  - Description
* - `!`
  - NOT
* - `&`
  - AND
* - `&&`
  - Short-circuiting AND
* - `|`
  - OR
* - `||`
  - Short-circuiting OR
* - `^`
  - XOR
:::

:::{list-table} Comparison operators
:name: tab-comp-op
:header-rows: 1
:align: center

* - Operator
  - Description
* - `!=`
  - Nonequality
* - `<`
  - Less than
* - `<=`
  - Less than or equal
* - `==`
  - Equality
* - `>`
  - Greater than
* - `>=`
  - Greater than or equal
:::
