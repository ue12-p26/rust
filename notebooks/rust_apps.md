# Notable Rust applications

We present in {numref}`tab-rust-apps` a list of some useful Rust
applications.
Most of them are replacement for standard Unix commands that run *faster*.

:::{list-table} Useful Rust applications
:name: tab-rust-apps
:header-rows: 1
:align: center

* - Crate
  - Command
  - Description
* - `bat`
  - `bat`
  - Replacement for `cat`.
* - `bottom`
  - `btm`
  - Replacement for `top`.
* - `dust`
  - `dust`
  - Replacement for `du` and `ncdu`.
* - `eza`
  - `eza` / `exa`
  - Replacement for `ls`.
* - `fd-find`
  - `fd`
  - Replacement for `find`.
* - `procs`
  - `procs`
  - Replacement for `ps`.
* - `ripgrep`
  - `rg`
  - Replacement for `grep`.
* - `minijinja-cli`
  - `minijinja-cli`
  - Rendering of MiniJinja/Jinja2 templates.
* - `rustlings`
  - `rustlings`
  - An application to learn Rust through exercises.
:::
