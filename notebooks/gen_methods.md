# Generic methods

:::{danger} Draft
This section is still being written.
:::

Implementing a method for all realisations of a generic struct:

```{code-cell} rust
struct Point<T> {
    x: T,
    y: T,
}

impl<T> Point<T> {
    fn x(&self) -> &T {
        &self.x
    }
}
```

It is possible to implement a method only for one realisation of a
generic:

```{code-cell} rust
impl Point<f32> {
    fn distance_from_origin(&self) -> f32 {
        (self.x.powi(2) + self.y.powi(2)).sqrt()
    }
}
```
